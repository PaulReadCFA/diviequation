Dividend valuation equations

There are three models to equity valuation based on the present value of expected dividends.

Constant Dividend (Equation 10):

PV t = Div t + 1 r

Constant Dividend Growth (Equation 14):

PV t = Div t ( 1 + g ) r - g = Div t + 1 r - g

Changing Dividend Growth (Equation 15):

PV 0 = ∑ i = 1 n Div i ( 1 + g 1 ) i ( 1 + r ) i + ∑ j = n + 1 ∞ Div j ( 1 + g 2 ) j ( 1 + r ) j
